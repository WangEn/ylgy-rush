羊了个羊网页版刷次数

## 使用说明

打开羊羊小程序，点击俺的名片，把 ID 记下来，在下面方输入自己的 ID ， 点击获取，将自动填写Token。

当然你也可以选择手动填写Token，Token的获取需要抓包。

填写要刷的次数，点击开始，即可开始刷次数。  

## 部署

### 服务器部署

```bash
npm run build
node .output/server/index.mjs 
```

### Vercel

```bash
vercel --prod
```
